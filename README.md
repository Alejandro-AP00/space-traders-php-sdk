## Installation

You can install the package via composer:

```bash
composer require alejandro-ap00/space-traders-php-sdk
```

## Usage

```php
$skeleton = new AlejandroAPorras\SpaceTraders();
echo $skeleton->echoPhrase('Hello, AlejandroAPorras!');
```
